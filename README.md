# Portafolio Personal con HTML, CSS y GitHub

## Descripción del proyecto

Esta página web utiliza todo lo que hemos aprendido hasta el momento acerca de HTML y CSS en clases.
La estructura del sitio es la siguiente:

* Head
    * Link al archivo CSS
    * Link externo a fuentes de *Google Fonts*
* Body
    * Header
        * Navbar
            * Logo
            * Links internos
    * Main
        * Sección **acerca de**
            * Artículo
            * *Aside*
        * Sección **habilidades**
            * Tabla de habilidades
            * Lista desordenada
            * Lista ordenada
        * Sección **proyectos**
            * CSS Grid con proyectos
        * Sección **contacto**
            * Formulario de contacto
            * *Aside*
    * Footer
        * Logo
        * Links en imágenes

## Tecnologías usadas

### CSS Flexbox

Se utiliza para alinear elementos del header y los *asides.*

### CSS Grid

Se ocupa para definir el *layout* general del sitio y para mostrar la galería de proyectos.

### GitHub

Aloja el código del sitio web y permite controlar versiones del proyecto.

## Ejecutar

Abrir el archivo index.html en un navegador web.

![Captura del sitio, incluyendo el navbar](/assets/nav.png)

Para ir hacia una sección en específico, se recomienda utilizar los enlaces de la barra de navegación.

![Formulario de contacto](/assets/form.png)

Para rellenar el formulario de contacto, hacer clic en cada zona e ingresar el texto requerido.
Al finalizar, presionar el botón de **Enviar** para enviar el formulario.
